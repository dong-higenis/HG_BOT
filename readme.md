## HG BOT
### 소형 로봇용 컨트롤 보드류

### 보드 리스트 및 설명
1. STM32G4-WT-CAN
    - STM32G4 무게 측정 보드

2. STM32G4-LC
    - STM32G4 NeoPixel x2, Servo x2 제어 보드

3. STM32G4-IMU-CAN
    - STM32G4 IMU 보드

4. STM32G4-DMC
    - STM32G4 DC Motor x4 제어 보드
    - 4채널 중 2채널은 엔코더 입력 가능

5. ESP32-C3-CAN
    - ESP32C3 기반 CAN 통신 모듈
